# Canvas
Canvas is a light web-based sketchpad to make quick sketches and collages that can then be shared or saved. The project is designed to be easy to use on a mouse or trackpad.

## Features
- Drawing Tools:
    - Circles
    - Rectangles
    - Polygons
    - Lines
    - Curves
    - Pen
- Editing Tools:
    - Fill/Stroke Picker
    - Eraser
    - Paint Bucket
    - Eye Dropper
    - Lasso
    - Undo/Redo
    - Clear Canvas
- File Management:
    - Save As
    - Share
